# MyRealTrip 해외렌터카 AI 생성 콘텐츠

마이리얼트립 해외렌터카 신규 버티컬 서비스에 내재화할 **AI 기반 여행자 가이드 콘텐츠**를 정리한 리포지토리입니다.

---

## 배경

2026년 상반기 오픈한 마이리얼트립 해외렌터카 서비스는 기존 시장의 "공급자 중심" 구조를 **"소비자 중심"** 으로 전환하는 것을 핵심 원칙으로 삼았습니다. 특히 여행자가 겪는 23개 Pain Point 중 다수가 **"현지에서 필요한 정보가 부족하다"** 는 문제에 뿌리를 두고 있어, 여행 준비·이용·사후 전 단계에 걸친 **국가/도시별 상세 가이드 콘텐츠**가 필수적으로 요구되었습니다.

하지만 30개국 이상, 200개 이상의 도시에 대한 콘텐츠를 기존 방식(작가/에디터 리소스 투입)으로 제작하는 것은 시간·비용 관점에서 실질적으로 불가능한 스코프였습니다. 이 문제를 **AI 기반 콘텐츠 생성 + 검수 파이프라인**으로 해결한 결과물이 본 리포지토리의 파일입니다.

---

## 리포지토리 구성

```
ai-generated-content/
└── overseas-rental-car/
    └── 여행_통합_가이드.html   # AI로 생성한 통합 여행 가이드 (인터랙티브 웹 페이지)
```

### `여행_통합_가이드.html`

단일 HTML 파일 안에 **여행 전 준비 → 여행 중 이용 → 여행 후 후기 작성**까지 렌터카 여행자의 전체 여정을 커버하는 인터랙티브 가이드입니다.

**주요 섹션**

| 섹션 | 내용 |
| --- | --- |
| **여행 전 준비** | 국제운전면허증, 여행 준비물, 국가별 요건, 예약 확인 체크리스트 |
| **여행 중 드라이브 가이드** | 국가/도시별 속도 제한, 교통 규칙, 주유 방법, 주차 규정, 통행료(ETC/톨게이트) 안내 |
| **긴급 상황 대응** | 사고·고장·분실 시 대응 절차, 현지 긴급 연락처, 3자 통역 안내 |
| **이용 후기 작성** | 실제 배차 차량, 만족도(전체·세부), 추가 비용·영업 시도 등 데이터 자산화용 후기 폼 |

**커버 지역**

- **아시아**: 일본(도쿄·오사카·후쿠오카·오키나와·삿포로·교토·나고야 등 30+ 도시), 대만(타이베이·가오슝), 동남아·인도, 대마도, 괌·사이판
- **북미**: 미국(뉴욕·LA·라스베이거스·마이애미·시애틀 등), 캐나다, 하와이
- **유럽**: 이탈리아(로마·나폴리), 스페인(마드리드·세비야·그라나다·그란카나리아), 독일, 영국(런던), 프랑스(니스), 오스트리아(그라츠), 그리스
- **오세아니아**: 호주(골드코스트·다윈·뉴캐슬), 뉴질랜드
- **중남미·중동**: 멕시코, 두바이

총 **200+ 도시** 데이터 세트 포함.

**인터랙티브 기능**

- 국가/도시 선택 시 해당 지역 맞춤 정보 동적 렌더링
- 예약된 도시 자동 필터링 (예약 정보 연동 시)
- 체크리스트 진행 상태 저장 (LocalStorage)
- 후기 작성 폼 → 실제 배차 차종 통계, 추가비용 발생 여부 등 자체 데이터 수집 로직

## AI 활용 방식

본 콘텐츠는 다음 파이프라인으로 생성되었습니다.

1. **스펙 정의**: 렌터카 여행자의 여정 단계별 필요 정보 카테고리 정의 (여행 전/중/후 × 국가별 교통규칙·통행료·주유 방식 등)
2. **AI 프롬프팅**: 도시별 구조화된 프롬프트로 규칙·속도·주유·주차·통행료 데이터 대량 생성
3. **크로스 체크**: 공식 관광청·렌터카 파트너사 자료와 대조하여 사실 검증
4. **HTML 인터랙션 레이어**: AI 코드 어시스턴트(Claude Code)로 도시 선택·필터링·체크리스트 저장 등 프론트엔드 로직 자동 생성
5. **단일 파일 통합**: 별도 서버 없이 브라우저에서 즉시 동작하는 정적 HTML 단일 파일로 패키징

**결과**

- 기존 에디토리얼 방식 대비 **콘텐츠 제작 리소스 90%+ 절감**
- 30개국 200+ 도시 커버 (기존 방식 대비 커버리지 5~10배)
- 단일 HTML 배포 → CDN·서버 인프라 불필요
- 새 도시 추가 시 데이터만 추가하면 즉시 반영 가능한 구조

## 서비스 내재화 계획

본 콘텐츠는 다음 지점에 통합됩니다.

- **예약 확정 후**: 바우처와 함께 여행 전 준비 체크리스트 발송 (D-7, D-1 리마인드)
- **여행 중**: 앱 내 "내 여행" 탭에서 예약 도시 자동 필터링된 드라이브 가이드 노출
- **여행 종료 후**: 후기 작성 폼으로 배차 통계·현지 추가비용 데이터 수집 → 다음 여행자에게 인사이트로 제공 (데이터 선순환 파이프라인)
